# 🗄️ Trabalho 03 — Consultas SQL e Extração de Dados

Este trabalho faz parte da disciplina **Introdução à Visualização de Dados e SQL** e tem como objetivo a prática de **consultas em bancos de dados relacionais utilizando SQL (ANSI)**.

---

## 🎯 Objetivo da atividade

Desenvolver consultas SQL capazes de **extrair e filtrar informações de tabelas relacionais**, aplicando condições lógicas para simular cenários reais de análise de dados em empresas.

As consultas exploram conceitos fundamentais de SQL, como:

* Seleção de colunas com `SELECT`
* Filtragem de dados com `WHERE`
* Operadores lógicos (`AND`, `OR`)
* Comparações entre valores numéricos, texto e datas

---

## 📁 Estrutura do banco de dados

Para a realização dos exercícios foi utilizado um banco de dados fictício da **TechStore**, composto por quatro tabelas principais:

* **produtos** — informações sobre itens vendidos na loja
* **clientes** — dados cadastrais dos clientes
* **pedidos** — registros de compras realizadas
* **vendedores** — desempenho mensal da equipe de vendas

Essas tabelas permitem simular cenários comuns de análise empresarial, como controle de estoque, comportamento de clientes, auditoria de pedidos e avaliação de desempenho comercial.

---

## 🛠️ Consultas desenvolvidas

Durante a atividade foram desenvolvidas consultas SQL para diferentes objetivos, incluindo:

* Listagem completa de dados de uma tabela
* Filtragem de produtos por categoria, preço e estoque
* Identificação de clientes por localização e tipo de plano
* Análise de pedidos com base em valor, status e data
* Seleção de vendedores com base em região, metas e volume de vendas

As consultas aplicam **combinações de filtros lógicos e comparações**, permitindo extrair informações específicas do banco de dados.

---

## 🧠 Contexto do exercício

Os cenários propostos simulam situações reais dentro de uma empresa de tecnologia, incluindo:

* Auditoria de estoque
* Planejamento de campanhas de marketing
* Análise regional de clientes
* Monitoramento financeiro de pedidos
* Avaliação de desempenho de vendedores

Essas atividades demonstram como o SQL pode ser utilizado para **transformar dados brutos em informações úteis para tomada de decisão**.

---

## 🚀 Considerações finais

Este trabalho reforça a importância do **SQL como ferramenta fundamental para manipulação e análise de dados** em bancos relacionais.

A prática com consultas e filtros permite compreender como extrair informações relevantes de grandes volumes de dados, habilidade essencial para áreas como:

* Análise de dados
* Engenharia de dados
* Business Intelligence

O exercício contribui para o desenvolvimento da lógica de consulta e da capacidade de estruturar análises diretamente no banco de dados.
