# 📊 Introduction to Data Visualization and SQL

This repository contains the **assignments and exercises developed in the course _Introduction to Data Visualization and SQL_**, taken at college, with a focus on **data analysis, exploration, and visualization**, as well as the progressive learning of the **SQL** language.

The course follows a practical approach, connecting data to decision-making through interactive dashboards and database queries, using tools widely adopted in the industry.

---

## 🎯 Course Objective

Develop fundamental skills for the data field, including:

- Data organization and exploration  
- Creation of clear and informative visualizations  
- Data storytelling  
- Introduction to SQL for data analysis  
- Extracting relevant insights from structured datasets  

---

## 🛠️ Tools and Technologies Used

- **Google Looker Studio**  
- **CSV files**  
- **Google Sheets**  
- **SQL** (progressive learning)  
- **Data Visualization** and **Data Analysis** concepts
