# 📊 Trabalho 01 — Dashboard de Exploração de Dados Diversos

Este trabalho faz parte da disciplina **Introdução à Visualização de Dados e SQL** e tem como objetivo estabelecer a base para a análise e exploração de dados por meio da criação de um **dashboard interativo no Google Looker Studio**.

---

## 🎯 Objetivo da atividade

Criar um dashboard inicial que permita a **exploração visual de dados**, utilizando diferentes fontes e tipos de visualização, de forma organizada, clara e justificada.

---

## 📁 Fontes de dados utilizadas

- Arquivos **CSV**
- **Google Sheets**

Os dados foram corretamente conectados e mapeados no Google Looker Studio, garantindo consistência e coerência na análise.

---

## 📊 Visualizações desenvolvidas

O dashboard contém:

- Tabelas para exibição estruturada dos dados
- Pelo menos **duas visualizações distintas para cada fonte de dados**
- Gráficos interativos para facilitar a análise e interpretação das informações

Entre os elementos utilizados, destacam-se:
- Gráficos de pizza
- Scorecard

---

## 🧠 Contexto do exercício

Como cenário de análise, foi utilizado um **cardápio fictício de uma lanchonete**, permitindo a visualização de informações como:

- Descrição do produto 
- Sabor de cada lanche  
- Tipo de lanche 
- Quantidade de vendas

Esses dados foram escolhidos para demonstrar como a visualização adequada pode auxiliar na **análise e tomada de decisões**.

---

## 🛠️ Ferramentas utilizadas

- Google Looker Studio  
- Arquivos CSV  
- Google Sheets
