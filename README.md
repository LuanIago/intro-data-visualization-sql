# 📊 Introdução à Visualização de Dados e SQL

Este repositório reúne os **trabalhos e exercícios desenvolvidos na disciplina _Introdução à Visualização de Dados e SQL_**, cursada na faculdade, com foco em **análise, exploração e visualização de dados**, além do aprendizado progressivo da linguagem **SQL**.

A disciplina propõe uma abordagem prática, conectando dados a decisões por meio de dashboards interativos e consultas a bases de dados, utilizando ferramentas amplamente adotadas no mercado.

---

## 🎯 Objetivo da disciplina

Desenvolver competências fundamentais para a área de dados, incluindo:

- Organização e exploração de dados
- Criação de visualizações claras e informativas
- Storytelling com dados
- Introdução à linguagem SQL para análise de dados
- Extração de insights relevantes a partir de bases estruturadas

---

## 🛠️ Ferramentas e tecnologias utilizadas

- **Google Looker Studio**
- **Arquivos CSV**
- **Google Sheets**
- **SQL** (em aprendizado progressivo)
- Conceitos de **Data Visualization** e **Data Analysis**
